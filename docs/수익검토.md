# TubeDeck 수익성 분석 (현실적)

> 환율 기준: 1 USD ≈ ₩1,450 (2026년 3월)
> 수익 모델: 광고 + 광고 제거 구매(일회성) + 기부(선택)

---

## 1. 현실 인식

### 이 앱이 뭔지부터 정리하자
- YouTube 구독 채널을 **폴더처럼 정리**하는 유틸리티 앱
- "없으면 불편하지만, 없어도 유튜브는 잘 돌아감"
- **충동 구매 유도 불가** — 생산성 도구도, 게임도, 콘텐츠도 아님
- 타겟: 구독 50개+ 헤비 유저 → 전체 YouTube 유저의 극소수

### 경쟁자: PocketTube
- 200,000+ 유저, Patreon 기부 모델
- **몇 년 운영해도 기부 기반** = 이 니치로 큰 돈 안 된다는 반증

## 2. 현실적 다운로드 예측

신규 니치 유틸리티 앱, 마케팅 예산 0원, 1인 개발 기준:

| 기간 | 예상 다운로드 | DAU | 근거 |
|------|-------------|-----|------|
| 출시 1개월 | 200~500 | 50~100 | ASO만으로 초기 유입. 거의 없음 |
| 3개월 | 1,000~3,000 | 100~300 | 리뷰 쌓이면서 서서히 노출 |
| 6개월 | 3,000~8,000 | 200~800 | 16개 언어 ASO 효과 발현 |
| 1년 | 5,000~15,000 | 300~1,500 | 운 좋으면. 바이럴 없으면 1만 이하 |

> **현실**: 마케팅 없는 니치 유틸리티 앱의 1년 다운로드는 **5,000~10,000이 평균**.
> 15,000도 꽤 잘 된 케이스.

## 3. 수익 시나리오

### A. 광고 수익 (AdMob 배너)

| 항목 | 현실적 수치 |
|------|-----------|
| DAU | 300~1,000 (1년차) |
| 배너 eCPM (글로벌 평균) | ₩500~₩1,000 |
| 유저당 일일 노출 | 3~5회 (유틸리티 앱은 체류 시간 짧음) |
| **일일 광고 수익** | ₩450 ~ ₩5,000 |
| **월 광고 수익** | **₩13,500 ~ ₩150,000** |

> 솔직히 말하면 **월 5~15만원** 수준. 커피값.

### B. 광고 제거 구매 (₩4,350 일회성)

| 항목 | 현실적 수치 |
|------|-----------|
| 1년 누적 다운로드 | 5,000~10,000 |
| 광고 제거 구매율 | 1~3% (니치 유틸리티 평균) |
| 구매자 수 | 50~300명 |
| **1년 구매 수익** | **₩217,500 ~ ₩1,305,000** |
| Google 수수료 (15%) | -₩32,625 ~ -₩195,750 |
| **실수령** | **₩184,875 ~ ₩1,109,250** |

> 1년 걸려서 **20~110만원**. 월 평균 **₩15,000 ~ ₩92,000**.

### C. 기부 (Buy Me a Coffee 등)

| 항목 | 현실적 수치 |
|------|-----------|
| 기부 참여율 | 0.1~0.5% (대부분 안 함) |
| 1년 기부자 수 | 5~50명 |
| 평균 기부 금액 | ₩3,000~₩5,000 |
| **1년 기부 수익** | **₩15,000 ~ ₩250,000** |

> 기부는 **보너스** 정도. 수익원으로 기대하면 안 됨.

## 5. 냉정한 결론

| 질문 | 답 |
|------|-----|
| 돈 되나? | **용돈 수준**. 월 3~15만원 (1년차) |
| 풀타임 가능? | **불가능** |
| 투자 대비 수익? | 개발 시간 생각하면 **최저시급 이하** |
| 그래도 할 가치? | 포트폴리오 + 부수입 + 경험치로는 OK |
| 성장 가능성? | 바이럴 터지면 x5~x10 가능하나, **니치 유틸리티에서 바이럴은 거의 안 일어남** |

### 수익이 의미 있어지려면
- DAU **5,000+** 필요 → 다운로드 **50,000+** 필요
- 마케팅 없이 달성하려면 **2~3년** 이상 소요
- YouTube/Reddit/커뮤니티 직접 홍보 필수

## 6. 추천 전략

1. **1단계**: 무료(광고 없이) 출시 → 리뷰 20~30개 확보
2. **2단계**: 배너 광고 도입 (배너만. 전면 광고는 유저 이탈 유발)
3. **3단계**: 광고 제거 일회성 구매 ₩4,350 추가
4. **선택사항**: 설정 화면에 기부 링크 (기대는 말 것)
5. **ASO 최적화**: 16개 언어 스토어 설명 꼼꼼히 — 유일한 무료 마케팅 수단
6. **API 비용 관리**: RSS 피드로 API 호출 최소화 (이미 RSSService 존재)

### 하지 말아야 할 것
- 구독 모델 도입 (이 앱에 구독료 낼 사람 없음)
- 전면 광고 남발 (유저 이탈 → 리뷰 폭락)
- 마케팅비 투자 (ROI 안 나옴)
