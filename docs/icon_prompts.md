# TubeDeck 앱 아이콘 AI 생성 프롬프트

## 컨셉 1: 비디오 카드 덱

### DALL-E / Midjourney
```
App icon design, three white video cards stacked at slight angles like a deck, front card has a red play button in center, red gradient background (#FF0000 to #CC0000), rounded square shape, modern flat design, subtle drop shadow, clean minimalist style, 512x512, high quality
```

### 한국어 설명
빨간색 그라데이션 배경 위에 3장의 흰색 카드가 살짝 기울어져 겹쳐있고, 맨 앞 카드 중앙에 빨간 재생 버튼이 있는 앱 아이콘

---

## 컨셉 2: 재생 버튼 + 부채꼴 덱

### DALL-E / Midjourney
```
App icon design, three cards spread in a fan shape (blue, yellow, white), large red circular play button overlapping in center, dark navy blue background (#1a1a2e), rounded square shape, modern design with depth, soft shadows, 512x512, high quality
```

### 한국어 설명
어두운 네이비 배경 위에 파란색/노란색/흰색 카드 3장이 부채꼴로 펼쳐져 있고, 중앙에 큰 빨간 원형 재생 버튼이 겹쳐진 앱 아이콘

---

## 컨셉 3: 미니멀 TV

### DALL-E / Midjourney
```
App icon design, minimalist white TV/monitor with dark screen and white play button, three small video thumbnail cards below the TV, red gradient background (#FF4444 to #CC0000), rounded square shape, flat design, clean and simple, 512x512, high quality
```

### 한국어 설명
빨간색 배경 위에 흰색 TV/모니터 프레임과 어두운 화면(재생 버튼 포함), 하단에 작은 영상 카드 3개가 있는 미니멀한 앱 아이콘

---

## 컨셉 4: 구독 피드 리스트

### DALL-E / Midjourney
```
App icon design, three horizontal video cards stacked vertically like a feed list, each card has red thumbnail on left with play icon and text lines on right, dark gray/black background (#212121), rounded square shape, YouTube-style layout, modern flat design, 512x512, high quality
```

### 한국어 설명
검정색 배경 위에 3개의 영상 카드가 세로로 쌓여있는 피드 리스트 형태, 각 카드는 왼쪽에 빨간 썸네일(재생 아이콘)과 오른쪽에 텍스트 라인이 있는 YouTube 스타일 앱 아이콘

---

## 공통 추가 키워드

```
- iOS app icon style
- Android adaptive icon
- No text on icon
- Professional design
- High contrast
- Recognizable at small sizes
```

## 권장 도구

| 도구 | 장점 |
|------|------|
| Midjourney | 고품질, 예술적 |
| DALL-E 3 | 프롬프트 정확도 높음 |
| Ideogram | 텍스트 처리 우수 |
| Leonardo.ai | 무료 크레딧 제공 |
| Canva AI | 간편한 편집 |

## 생성 후 처리

1. 512x512 또는 1024x1024로 생성
2. 배경 투명화 (필요시)
3. Android adaptive icon용 foreground/background 분리
4. 다양한 해상도로 리사이즈 (mdpi, hdpi, xhdpi, xxhdpi, xxxhdpi)
